# Tableau-Workbook
# 🌍 Tableau Data Visualisation Project — Data Technician Bootcamp

This repository contains my Tableau project completed during the **Data Technician Bootcamp**, where I explored global health, retail, and demographic datasets to build interactive dashboards and communicate insights through data storytelling.

The project demonstrates practical skills in **data visualisation**, **dashboard design**, and **analytical thinking**, using Tableau Public.

---

## 🚀 Project Overview

Across multiple worksheets and dashboards, I analysed trends in life expectancy, population distribution, and health metrics. The project focuses on:

- Building **interactive dashboards**
- Using **filters**, **slicers**, and **parameters** for user‑driven exploration  
- Creating **calculated fields** to derive new insights  
- Designing visuals that communicate clear, compelling stories  
- Applying a range of chart types including bar, line, pie, scatter, and filled maps  

---

## 📊 Key Visualisations

### **1. Life Expectancy by Continent (Bar Chart)**
- Shows average life expectancy for each continent  
- Filtered to the most recent year  
- Sorted to highlight disparities between regions  
- Supports quick comparison of global health outcomes  

### **2. Life Expectancy Trend (Line Chart)**
- Displays life expectancy over time for the **top 5 countries**  
- Uses a Top N filter based on average life expectancy  
- Colour‑coded lines for clarity and comparison  

### **3. Population Distribution by Gender (Pie Chart)**
- Focuses on the latest year for a selected country  
- Includes a **single‑value dropdown filter** for country selection  
- Uses percentage‑of‑total labels for clear interpretation  

### **4. Life Expectancy vs BMI (Scatter Plot)**
- Compares two key health metrics  
- Points represent individual countries  
- Colour‑coded by continent  
- Includes continent filter for deeper exploration  

### **5. Additional Visual of Choice**
- Demonstrates flexibility in chart selection and storytelling  
- Reinforces understanding of Tableau’s marks, filters, and formatting tools

- Cross‑sheet filters for seamless exploration
